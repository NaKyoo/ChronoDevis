# ChronoDevis - Un Générateur de Devis Simple et Rapide

**ChronoDevis** est un générateur de devis moderne, conçu pour être simple, rapide et efficace. Il permet de créer, gérer et exporter des devis professionnels en quelques clics.

![ChronoDevis Banner](/public/assets/img/chronodevis-web-app.png)

## Fonctionnalités

- **Création Simple de Devis :** Utilisez un formulaire intuitif pour générer rapidement des devis.
- **Sauvegarde et Historique :** Vos devis sont conservés directement dans votre navigateur.
- **Options d'Exportation :** Téléchargez en PDF ou envoyez directement par email.
- **Multi-langues :** Support complet du Français et de l'Anglais.
- **Format varié :** Exportez vos données en JSON, XLSX, CSV, et XML.

## Technologies Utilisées

- **Next.js 15**
- **TypeScript**
- **Tailwind CSS**
- **React Hook Form & Zod**
- **Shadcn UI**
- **React Email**

## Démarrage Rapide

### Prérequis

- Node.js et npm installés.

### Installation

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/NaKyoo/ChronoDevis.git
   cd ChronoDevis
   ```
2. Installez les dépendances :
   ```bash
   npm install
   ```
3. Lancez le serveur de développement :
   ```bash
   npm run dev
   ```
